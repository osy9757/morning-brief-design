# MorningBrief — 개인 투자 분석 서비스 설계 (기획·방법론)

이 저장소는 **LLM 기반 개인 투자 분석 서비스의 설계 문서 세트**다. 매일 아침 시장을 분석해 데일리 브리핑을 생성하고, 개인 포트폴리오에 장전 매매 시그널과 진입 타점을 제시하는 1인용 서비스를 어떻게 구조화했는지 담았다.

> 실제 구현 코드는 비공개 저장소에 있다. 여기는 **어떤 LLM이 구현해도 거의 동일한 결과가 나오도록** 모든 선택지를 값으로 고정한 기획·아키텍처만 공개한다.

## 핵심 설계 철학

원본 방법론(vibe-investing)에서 가져온 원칙:

- **LLM은 원칙적으로 숫자를 만들지 않는다.** 레짐 점수·팩터·시그널은 결정적 엔진이 계산하고, LLM은 그 숫자를 *해설*한다 — 할루시네이션을 구조적으로 차단.
- **예외: 진입 타점은 하이브리드.** 엔진이 기술 지표(가격·이평·ATR·지지/저항·52주 범위)를 계산해 주면, LLM이 그 데이터로 진입 구간을 *판단*하고 근거를 쓴다. 단 산출 가격은 관측 범위 가드를 통과해야 하며, 위반·키 없음 시 엔진 수식으로 폴백한다.
- **매매 액션은 규칙, LLM 아님.** 손절·낙폭 방어 같은 리스크 로직(R0~R7)은 결정적으로 유지.
- **프로바이더는 변수.** 모델·라우팅을 설정에서 교체 가능(멀티-LLM), 키 없으면 해당 섹션만 graceful degraded.

## 분석 방법론 (요약)

- **5-Factor 침체 컴포지트 → 4-Phase 레짐**: 수익률곡선·Sahm·경기활동·선행지표·신용스트레스로 시장 국면 분류.
- **섹터·자금흐름·Fear&Greed**: 중급 투자자용 지표 밀도, 자산군 로테이션.
- **종목 팩터(US/Domestic Score)**: 모멘텀·밸류·퀄리티/성장·저변동성·매크로적합·리스크이벤트 가중 합산.
- **멀티-LLM 위원회**: Bull/Bear/Risk/PM 다각 판정 후 확신도 산출.
- **포트폴리오 시그널 R0~R7**: Tier4 하드룰·손절·불타기·물타기·리밸런싱 + 진입 타점.

## 면책

개인 분석용 설계 문서이며 투자 권유가 아니다. 모든 추천/시그널은 사용자 본인의 판단·책임이다.
